Dependencies
============
1. MongoDB
    - navigate to MongoDB's main website and download the non-enterprise version of MongoDB
    - make sure local DB is running on port `27017`
2. pymongo
    - on your machine install pymongo with
        Mac: `pip3 install pymongo`
        Windows: `pip install pymongo`

Run Instructions
================
1. Add any sensitive information that you might have to sensitive_info.csv in a pipe delimited format.
    Example: 1234 Durant Avenue|1234567|123-456-789
   
2. To run on the machine, open up a terminal and run the command
    `python3 labeller.py`
